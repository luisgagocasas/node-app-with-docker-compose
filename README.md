# Node and Docker-compose

docker-compose structure for nodejs. a simple server with express and nodemon.

## Software
- docker
- docker-compose
- express
- nodemon

## Usage

UP

```bash
docker-compose up -d
```

DOWN

```bash
docker-compose down
```

LOGS

```bash
docker-compose logs -f
```

| Service      | Path                    |
| ------------ | ----------------------- |
